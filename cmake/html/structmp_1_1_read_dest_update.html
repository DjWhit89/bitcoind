Detailed Description

template<typename Value>
struct mp::ReadDestUpdate< Value >

Destination parameter type that can be passed to ReadField function as an alternative to ReadDestEmplace. Instead of requiring an emplace callback to construct a new value, it just takes a reference to an existing value and assigns a new value to it.
