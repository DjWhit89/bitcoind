\doxysubsection{Detailed Description}
Metadata describing a serialized version of a UTXO set from which an assumeutxo \doxyref{Chainstate}{p.}{class_chainstate} can be constructed. All metadata fields come from an untrusted file, so must be validated before being used. Thus, new fields should be added only if needed.
