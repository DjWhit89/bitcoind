\doxysubsection{Detailed Description}
Optimized Sip\+Hash-\/2-\/4 implementation for \doxyref{uint256}{p.}{classuint256}.

This class caches the initial Sip\+Hash v[0..3] state derived from (k0, k1) and implements a specialized hashing path for \doxyref{uint256}{p.}{classuint256} values, with or without an extra 32-\/bit word. The internal state is immutable, so \doxyref{Presalted\+Sip\+Hasher}{p.}{class_presalted_sip_hasher} instances can be reused for multiple hashes with the same key.
