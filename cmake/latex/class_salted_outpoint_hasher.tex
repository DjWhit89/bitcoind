Having the hash noexcept allows libstdc++\textquotesingle{}s unordered\+\_\+map to recalculate the hash during rehash, so it does not have to cache the value. This reduces node\textquotesingle{}s memory by sizeof(size\+\_\+t). The required recalculation has a slight performance penalty (around 1.\+6\%), but this is compensated by memory savings of about 9\% which allow for a larger dbcache setting.
