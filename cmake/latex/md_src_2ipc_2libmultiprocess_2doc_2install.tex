\chapter{libmultiprocess Installation }
\label{md_src_2ipc_2libmultiprocess_2doc_2install}\index{libmultiprocess Installation@{libmultiprocess Installation}}
\label{md_src_2ipc_2libmultiprocess_2doc_2install_autotoc_md19}%


Installation currently requires Cap\textquotesingle{}n Proto 0.\+7 or higher\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{apt\ install\ libcapnp-\/dev\ capnproto}
\DoxyCodeLine{brew\ install\ capnp\ cmake}
\DoxyCodeLine{dnf\ install\ capnproto}

\end{DoxyCode}


Installation steps are\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{mkdir\ build}
\DoxyCodeLine{cd\ build}
\DoxyCodeLine{cmake\ ..}
\DoxyCodeLine{make}
\DoxyCodeLine{make\ check\ \#\ Optionally\ build\ and\ run\ tests}
\DoxyCodeLine{make\ install}

\end{DoxyCode}


To build with libmultiprocess in a CMake project can specify\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{find\_package(Libmultiprocess)}
\DoxyCodeLine{target\_capnp\_sources(mytarget\ \$\{CMAKE\_CURRENT\_SOURCE\_DIR\}\ myschema.capnp)}

\end{DoxyCode}


Which will locate the libmultiprocess cmake package, and call the {\ttfamily target\+\_\+capnp\+\_\+sources} function to generate C++ files and link them into a library or executable target. See {\ttfamily example/\+CMake\+Lists.\+txt} for a complete example.

To build with libmultiprocess in a non-\/\+CMake project can use installed {\ttfamily $<$\doxyref{prefix}{p.}{rest_8cpp_a5b41c5ae4505891e6c53e26df197e02b}$>$/include/mpgen.mk} Makefile rule to generate C++ files, and {\ttfamily $<$\doxyref{prefix}{p.}{rest_8cpp_a5b41c5ae4505891e6c53e26df197e02b}$>$/lib/pkgconfig/libmultiprocess.pc} pkg-\/config definition to link against the runtime library.

For cross-\/compilation, it may be useful to build the runtime library and code generation binaries separately, which can be done with\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{make\ install-\/bin\ \#\ install\ bin/mpgen\ and\ related\ files}
\DoxyCodeLine{make\ install-\/lib\ \#\ install\ lib/libmultiprocess.a\ and\ related\ files}

\end{DoxyCode}


It is also possible to import CMake targets separately with\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{find\_package(Libmultiprocess\ COMPONENTS\ Bin)}
\DoxyCodeLine{find\_package(Libmultiprocess\ COMPONENTS\ Lib)}

\end{DoxyCode}
