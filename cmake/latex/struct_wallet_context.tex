\doxysubsection{Detailed Description}
\doxyref{Wallet\+Context}{p.}{struct_wallet_context} struct containing references to state shared between \doxyref{CWallet}{p.}{class_c_wallet} instances, like the reference to the chain interface, and the list of opened wallets.

Future shared state can be added here as an alternative to adding global variables.

The struct isn\textquotesingle{}t intended to have any member functions. It should just be a collection of state pointers that doesn\textquotesingle{}t pull in dependencies or implement behavior.
