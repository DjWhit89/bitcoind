\doxysubsection{Detailed Description}
256-\/bit opaque blob. \begin{DoxyNote}{Note}
This type is called \doxyref{uint256}{p.}{classuint256} for historical reasons only. It is an opaque blob of 256 bits and has no integer operations. Use \doxyref{arith\+\_\+uint256}{p.}{classarith__uint256} if those are required. 
\end{DoxyNote}
