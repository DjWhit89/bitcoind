\textbf{ Hash\+Writer} Tagged\+Hash (\begin{DoxyParamCaption}\item[{const std\+::string \&}]{tag}{}\end{DoxyParamCaption})}

Return a \doxyref{Hash\+Writer}{p.}{class_hash_writer} primed for tagged hashes (as specified in BIP 340).

The returned object will have SHA256(tag) written to it twice (= 64 bytes). A tagged hash can be computed by feeding the message into this object, and then calling \doxyref{Hash\+Writer\+::\+Get\+SHA256()}{p.}{class_hash_writer_ac641c79418391703ae971908c58b9185}.
