class bitdeque$<$ BITS\+\_\+\+PER\+\_\+\+WORD $>$}
Class that mimics std\+::deque$<$bool$>$, but with std\+::vector$<$bool$>$\textquotesingle{}s bit packing.

BITS\+\_\+\+PER\+\_\+\+WORD selects the (minimum) number of bits that are allocated at once. Larger values reduce the asymptotic memory usage overhead, at the cost of needing larger up-\/front allocations. The default is 4096 bytes.
