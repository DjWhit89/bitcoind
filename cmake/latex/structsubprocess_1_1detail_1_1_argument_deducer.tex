\doxysubsection{Detailed Description}
A helper class to \doxyref{Popen}{p.}{classsubprocess_1_1_popen} class for setting options as provided in the \doxyref{Popen}{p.}{classsubprocess_1_1_popen} constructor. This design allows us to {\itshape not} have any fixed position to any arguments and specify them in a way similar to what can be done in python.
