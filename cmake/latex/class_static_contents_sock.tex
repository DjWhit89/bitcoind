\doxysubsection{Detailed Description}
A mocked \doxyref{Sock}{p.}{class_sock} alternative that returns a statically contained data upon read and succeeds and ignores all writes. The data to be returned is given to the constructor and when it is exhausted an EOF is returned by further reads.
