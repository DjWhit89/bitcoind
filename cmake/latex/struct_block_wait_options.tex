\textbf{ CAmount} fee\+\_\+threshold \{MAX\+\_\+\+MONEY\}}

The wait method will not return a new template unless it has fees at least fee\+\_\+threshold sats higher than the current template, or unless the chain tip changes and the previous template is no longer valid.

A caller may not be interested in templates with higher fees, and determining whether fee\+\_\+threshold is reached is also expensive. So as an optimization, when fee\+\_\+threshold is set to MAX\+\_\+\+MONEY (default), the implementation is able to be much more efficient, skipping expensive checks and only returning new templates when the chain tip changes. \index{BlockWaitOptions@{BlockWaitOptions}!timeout@{timeout}}
