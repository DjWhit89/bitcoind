bool Signals\+Opt\+In\+RBF (\begin{DoxyParamCaption}\item[{const \textbf{ CTransaction} \&}]{tx}{}\end{DoxyParamCaption})}

Check whether the sequence numbers on this transaction are signaling opt-\/in to replace-\/by-\/fee, according to BIP 125. Allow opt-\/out of transaction replacement by setting n\+Sequence $>$ MAX\+\_\+\+BIP125\+\_\+\+RBF\+\_\+\+SEQUENCE (SEQUENCE\+\_\+\+FINAL-\/2) on all inputs.

SEQUENCE\+\_\+\+FINAL-\/1 is picked to still allow use of n\+Lock\+Time by non-\/replaceable transactions. All inputs rather than just one is for the sake of multi-\/party protocols, where we don\textquotesingle{}t want a single party to be able to disable replacement by opting out in their own input.
