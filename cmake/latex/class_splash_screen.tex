\doxysubsection{Detailed Description}
Class for the splashscreen with information of the running client.

\begin{DoxyNote}{Note}
this is intentionally not a QSplash\+Screen. Bitcoin Core initialization can take a long time, and in that case a progress window that cannot be moved around and minimized has turned out to be frustrating to the user. 
\end{DoxyNote}
