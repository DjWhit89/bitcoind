class\+: \doxyref{Buffer}{p.}{classsubprocess_1_1_buffer} This class is a very thin wrapper around std\+::vector$<$char$>$ This is basically used to determine the length of the actual data stored inside the dynamically resized vector.

This is what is returned as the output to the communicate function, so, users must know about this class.
