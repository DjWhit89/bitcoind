\doxysubsection{Detailed Description}
Used to relay blocks as header + vector$<$merkle branch$>$ to filtered nodes.

NOTE\+: The class assumes that the given \doxyref{CBlock}{p.}{class_c_block} has {\itshape at least} 1 transaction. If the \doxyref{CBlock}{p.}{class_c_block} has 0 txs, it will hit an assertion.
