\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \textbf{ T}$>$\newline
class Random\+Mixin$<$ T $>$}
Mixin class that provides helper randomness functions.

Intended to be used through CRTP\+: {\texttt{https\+://en.\+cppreference.\+com/w/cpp/language/crtp}}. An RNG class Funky\+RNG would derive publicly from \doxyref{Random\+Mixin$<$\+Funky\+RNG$>$}{p.}{class_random_mixin}. This permits \doxyref{Random\+Mixin}{p.}{class_random_mixin} from accessing the derived class\textquotesingle{}s rand64() function, while also allowing the derived class to provide more.

The derived class must satisfy the \doxyref{Random\+Number\+Generator}{p.}{concept_random_number_generator} concept.
