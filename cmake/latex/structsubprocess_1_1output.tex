\doxysubsection{Detailed Description}
Option to specify the output channel for the child process. It can be\+:
\begin{DoxyEnumerate}
\item An already open file descriptor.
\item A file name.
\item \doxyref{IOTYPE}{p.}{namespacesubprocess_aa7bbbf9cb819b8c3d488e0cf08a00b0c}. Usually a PIPE.
\end{DoxyEnumerate}
