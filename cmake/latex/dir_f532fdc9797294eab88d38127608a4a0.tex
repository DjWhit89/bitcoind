\doxysubsection{Detailed Description}
Simple C module for constant-\/time AES encryption and decryption.

Features\+:
\begin{DoxyItemize}
\item Simple, pure C code without any dependencies.
\item No tables or data-\/dependent branches whatsoever, but using bit sliced approach from {\texttt{https\+://eprint.\+iacr.\+org/2009/129.\+pdf}}.
\item Very small object code\+: slightly over 4k of executable code when compiled with -\/Os.
\item Slower than implementations based on precomputed tables or specialized instructions, but can do $\sim$15 MB/s on modern CPUs.
