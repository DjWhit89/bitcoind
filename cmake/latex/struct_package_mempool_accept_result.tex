Map from wtxid to finished Mempool\+Accept\+Results. The client is responsible for keeping track of the transaction objects themselves. If a result is not present, it means validation was unfinished for that transaction. If there was a package-\/wide error (see result in m\+\_\+state), m\+\_\+tx\+\_\+results will be empty.
