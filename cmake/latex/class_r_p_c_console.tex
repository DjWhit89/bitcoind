Split shell command line into a list of arguments and optionally execute the \doxyref{command(s)}{p.}{bitcoin-wallet_8cpp_a64b0c5844a38fc862dcb5b4e30b1ca49}. Aims to emulate {\ttfamily bash} and friends.


\begin{DoxyItemize}
\item Command nesting is possible with parenthesis; for example\+: validateaddress(getnewaddress())
\item Arguments are delimited with whitespace or comma
\item Extra whitespace at the beginning and end and between arguments will be ignored
\item Text can be "{}double"{} or \textquotesingle{}single\textquotesingle{} quoted
\item The backslash {\ttfamily \textbackslash{}} is used as escape character
\begin{DoxyItemize}
\item Outside quotes, any character can be escaped
\item Within double quotes, only escape {\ttfamily "{} and backslashes before a \textbackslash{}c "{}} or another backslash
\item Within single quotes, no escaping is possible and no special interpretation takes place
\end{DoxyItemize}
\end{DoxyItemize}
