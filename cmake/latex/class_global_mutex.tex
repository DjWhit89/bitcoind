\doxysubsection{Detailed Description}
Different type to mark \doxyref{Mutex}{p.}{sync_8h_a326d1d46ce05b5316d8ebe2694a10a6e} at global scope

Thread safety analysis can\textquotesingle{}t handle negative assertions about mutexes with global scope well, so mark them with a separate type, and eventually move all the mutexes into classes so they are not globally visible.

See\+: {\texttt{https\+://github.\+com/bitcoin/bitcoin/pull/20272\#issuecomment-\/720755781}}
