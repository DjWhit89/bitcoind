\doxysubsection{Detailed Description}
Vat id for server side of connection. Required argument to Rpc\+System\+::boot\+Strap()

"{}\+Vat"{} is Cap\textquotesingle{}n Proto nomenclature for a host of various objects that facilitates bidirectional communication with other vats; it is often but not always 1-\/1 with processes. Cap\textquotesingle{}n Proto doesn\textquotesingle{}t reference clients or servers per se; instead everything is just a vat.
