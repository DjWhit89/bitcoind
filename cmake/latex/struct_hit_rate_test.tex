The normalized hit rate for a given load.

The semantics are a little confusing, so please see the below explanation.

Examples\+:


\begin{DoxyEnumerate}
\item at load 0.\+5, we expect a perfect hit rate, so we multiply by 1.\+0
\item at load 2.\+0, we expect to see half the entries, so a perfect hit rate would be 0.\+5. Therefore, if we see a hit rate of 0.\+4, 0.\+4$\ast$2.0 = 0.\+8 is the normalized hit rate.
\end{DoxyEnumerate}
