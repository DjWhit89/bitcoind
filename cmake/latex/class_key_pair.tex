\doxysubsection{Detailed Description}
\doxyref{Key\+Pair}{p.}{class_key_pair}

Wraps a {\ttfamily \doxyref{secp256k1\+\_\+keypair}{p.}{structsecp256k1__keypair}} type, an opaque data structure for holding a secret and public key. This is intended for BIP340 keys and allows us to easily determine if the secret key needs to be negated by checking the parity of the public key. This class primarily intended for passing secret keys to libsecp256k1 functions expecting a {\ttfamily \doxyref{secp256k1\+\_\+keypair}{p.}{structsecp256k1__keypair}}. For all other cases, \doxyref{CKey}{p.}{class_c_key} should be preferred.

A \doxyref{Key\+Pair}{p.}{class_key_pair} can be created from a \doxyref{CKey}{p.}{class_c_key} with an optional merkle\+\_\+root tweak (per BIP342). See \doxyref{CKey\+::\+Compute\+Key\+Pair}{p.}{class_c_key_a1989d2094d3ab2510b9f839a1ab05b34} for more details.
