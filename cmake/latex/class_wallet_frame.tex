\doxysubsection{Detailed Description}
A container for embedding all wallet-\/related controls into \doxyref{Bitcoin\+GUI}{p.}{class_bitcoin_g_u_i}. The purpose of this class is to allow future refinements of the wallet controls with minimal need for further modifications to \doxyref{Bitcoin\+GUI}{p.}{class_bitcoin_g_u_i}, thus greatly simplifying merges while reducing the risk of breaking top-\/level stuff.
