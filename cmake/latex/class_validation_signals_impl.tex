\doxyref{Validation\+Signals\+Impl}{p.}{class_validation_signals_impl} manages a list of shared\+\_\+ptr$<$\+CValidation\+Interface$>$ callbacks.

A std\+::unordered\+\_\+map is used to track what callbacks are currently registered, and a std\+::list is used to store the callbacks that are currently registered as well as any callbacks that are just unregistered and about to be deleted when they are done executing.
