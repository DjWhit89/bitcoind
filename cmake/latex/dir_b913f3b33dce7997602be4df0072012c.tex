\doxysubsection{Detailed Description}
\doxysubsection{CI quick-\/reference}\label{dir_b913f3b33dce7997602be4df0072012c_autotoc_md13}
All CI is just bash and nix.


\begin{DoxyItemize}
\item {\bfseries{Workflow}}\+:
\begin{DoxyItemize}
\item {\ttfamily .github/workflows/ci.\+yml} – lists the jobs ({\ttfamily default}, {\ttfamily llvm}, …).
\end{DoxyItemize}
\item {\bfseries{Scripts}}\+:
\begin{DoxyItemize}
\item {\ttfamily ci/scripts/run.\+sh} – spins up the Nix shell then calls…
\item {\ttfamily ci/scripts/ci.\+sh} – …to configure, build, and test.
\end{DoxyItemize}
\item {\bfseries{Configuration}}\+:
\begin{DoxyItemize}
\item {\ttfamily ci/configs/$\ast$.sh} – defines flags for each job.
\item {\ttfamily shell.\+nix} – defines build environment (compilers, tools, libraries).
\end{DoxyItemize}
\item {\bfseries{Build directories}}\+:
\begin{DoxyItemize}
\item {\ttfamily build-\/$\ast$/} – separate build directories (like {\ttfamily build-\/default}, {\ttfamily build-\/llvm}) will be created for each job.
\end{DoxyItemize}
\end{DoxyItemize}

To run jobs locally\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{CI\_CONFIG=ci/configs/default.bash\ \ ci/scripts/run.sh}
\DoxyCodeLine{CI\_CONFIG=ci/configs/llvm.bash\ \ \ \ \ ci/scripts/run.sh}
\DoxyCodeLine{CI\_CONFIG=ci/configs/gnu32.bash\ \ \ \ ci/scripts/run.sh}
\DoxyCodeLine{CI\_CONFIG=ci/configs/sanitize.bash\ ci/scripts/run.sh}
\DoxyCodeLine{CI\_CONFIG=ci/configs/olddeps.bash\ \ ci/scripts/run.sh}

\end{DoxyCode}


By default CI jobs will reuse their build directories. {\ttfamily CI\+\_\+\+CLEAN=1} can be specified to delete them before running instead.
