\doxyref{Format}{p.}{struct_format} a value into a stream, delegating to operator$<$$<$ by default.

Users may override this for their own types. When this function is called, the stream flags will have been modified according to the format string. The format specification is provided in the range [fmt\+Begin, fmt\+End). For truncating conversions, ntrunc is set to the desired maximum number of characters, for example "{}\%.\+7s"{} calls format\+Value with ntrunc = 7.

By default, \doxyref{format\+Value()}{p.}{namespacetinyformat_a7de4f2eb18c53985cc3104817d6962bb} uses the usual stream insertion operator operator$<$$<$ to format the type T, with special cases for the c and p conversions. \index{tinyformat@{tinyformat}!makeFormatList@{makeFormatList}}
