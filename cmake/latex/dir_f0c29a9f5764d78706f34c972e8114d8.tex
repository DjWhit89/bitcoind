\doxysubsection{Detailed Description}
This directory contains the source code for the Bitcoin Core graphical user interface (GUI). It uses the [Qt]({\texttt{https\+://www1.\+qt.\+io/developers/}}) cross-\/platform framework.

The current precise version for Qt is specified in {\texttt{qt\+\_\+details.\+mk}}.\doxysubsubsection{Compile and run}\label{dir_f0c29a9f5764d78706f34c972e8114d8_autotoc_md101}
See build instructions\+: Unix, mac\+OS, Windows, Free\+BSD, Net\+BSD, Open\+BSD

When following your systems build instructions, make sure to install the {\ttfamily Qt} dependencies.

\doxysubsubsubsubsection{Setup Qt Creator}\label{dir_f0c29a9f5764d78706f34c972e8114d8_autotoc_md119}

\begin{DoxyEnumerate}
\item Make sure you\textquotesingle{}ve installed all dependencies specified in your systems build instructions
\item Follow the compile instructions for your system, adding the {\ttfamily -\/DCMAKE\+\_\+\+BUILD\+\_\+\+TYPE=Debug} build flag
\item Start Qt Creator. At the start page, do\+: {\ttfamily New} -\/$>$ {\ttfamily Import Project} -\/$>$ {\ttfamily Import Existing Project}
\item Enter {\ttfamily bitcoin-\/qt} as the Project Name and enter the absolute path to {\ttfamily src/qt} as Location
\item Check over the file selection, you may need to select the {\ttfamily forms} directory (necessary if you intend to edit $\ast$.ui files)
\item Confirm the {\ttfamily Summary} page
\item In the {\ttfamily Projects} tab, select {\ttfamily Manage Kits...}

{\bfseries{mac\+OS}}
\begin{DoxyItemize}
\item Under {\ttfamily Kits}\+: select the default "{}\+Desktop"{} kit
\item Under {\ttfamily Compilers}\+: select {\ttfamily "{}Clang (x86 64bit in /usr/bin)"{}}
\item Under {\ttfamily Debuggers}\+: select {\ttfamily "{}LLDB"{}} as debugger (you might need to set the path to your LLDB installation)
\end{DoxyItemize}

{\bfseries{Ubuntu \& Debian}}

Note\+: Some of these options may already be set
\begin{DoxyItemize}
\item Under {\ttfamily Kits}\+: select the default "{}\+Desktop"{} kit
\item Under {\ttfamily Compilers}\+: select {\ttfamily "{}GCC (x86 64bit in /usr/bin)"{}}
\item Under {\ttfamily Debuggers}\+: select {\ttfamily "{}GDB"{}} as debugger
\end{DoxyItemize}
\item While in the {\ttfamily Projects} tab, ensure that you have the {\ttfamily bitcoin-\/qt} executable specified under {\ttfamily Run}
\begin{DoxyItemize}
\item If the executable is not specified\+: click {\ttfamily "{}Choose..."{}}, navigate to {\ttfamily build/bin}, and select {\ttfamily bitcoin-\/qt}
\end{DoxyItemize}
\item You\textquotesingle{}re all set! Start developing, building, and debugging the Bitcoin Core GUI 
\end{DoxyEnumerate}
