\doxysubsection{Detailed Description}
Access to the wallet database. Opens the database and provides read and write access to it. Each read and write is its own transaction. Multiple operation transactions can be started using \doxyref{Txn\+Begin()}{p.}{classwallet_1_1_wallet_batch_a7a978d196dc95a23b3bf6dc6133b1f2f} and committed using \doxyref{Txn\+Commit()}{p.}{classwallet_1_1_wallet_batch_a22ec8139a504c485e397a00d38af0066} Otherwise the transaction will be committed when the object goes out of scope. Optionally (on by default) it will flush to disk on close. Every 1000 writes will automatically trigger a flush to disk.
