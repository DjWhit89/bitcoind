\doxysubsection{Detailed Description}
Option to specify the input channel for the child process. It can be\+:
\begin{DoxyEnumerate}
\item An already open file descriptor.
\item A file name.
\item \doxyref{IOTYPE}{p.}{namespacesubprocess_aa7bbbf9cb819b8c3d488e0cf08a00b0c}. Usual a PIPE
\end{DoxyEnumerate}

Eg\+: input\{PIPE\} OR in case of redirection, output of another \doxyref{Popen}{p.}{classsubprocess_1_1_popen} input\{popen.\+output()\}
