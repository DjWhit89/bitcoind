\doxysubsection{Detailed Description}
A convenience class for constructing the CCoins\+View$\ast$ hierarchy used to facilitate access to the UTXO set.

This class consists of an arrangement of layered \doxyref{CCoins\+View}{p.}{class_c_coins_view} objects, preferring to store and retrieve coins in memory via {\ttfamily m\+\_\+cacheview} but ultimately falling back on cache misses to the canonical store of UTXOs on disk, {\ttfamily m\+\_\+dbview}.
