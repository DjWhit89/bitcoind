\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \textbf{ T}$>$\newline
class Vec\+Deque$<$ T $>$}
Data structure largely mimicking std\+::deque, but using single preallocated ring buffer.


\begin{DoxyItemize}
\item More efficient and better memory locality than std\+::deque.
\item Most operations (\{push\+\_\+,pop\+\_\+,emplace\+\_\+,\}\{front,back\}(), operator[], ...) are O(1), unless reallocation is needed (in which case they are O(n)).
\item Supports \doxyref{reserve()}{p.}{class_vec_deque_a02348f13165f05921d779584b21d8228}, \doxyref{capacity()}{p.}{class_vec_deque_a260442ef344f843c480ff1f6aefd6db3}, \doxyref{shrink\+\_\+to\+\_\+fit()}{p.}{class_vec_deque_a5f16304f80b6fb253c7b0ead3e16dd18} like vectors.
\item No iterator support.
\item Data is not stored in a single contiguous block, so no data(). 
\end{DoxyItemize}
