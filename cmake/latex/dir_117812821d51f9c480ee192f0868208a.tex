\doxysubsection{Detailed Description}
\doxysubsection{Test library}\label{dir_117812821d51f9c480ee192f0868208a_autotoc_md231}
This contains files for the test library, which is used by the test binaries (unit tests, benchmarks, fuzzers, gui tests).

Generally, the files in this folder should be well-\/separated modules. New code should be added to existing modules or (when in doubt) a new module should be created.

The utilities in here are compiled into a library, which does not hold any state. However, the main file {\ttfamily setup\+\_\+common} defines the common test setup for all test binaries. The test binaries will handle the global state when they instantiate the {\ttfamily \doxyref{Basic\+Testing\+Setup}{p.}{struct_basic_testing_setup}} (or one of its derived classes).
