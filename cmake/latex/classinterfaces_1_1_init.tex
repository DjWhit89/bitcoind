\doxysubsection{Detailed Description}
Initial interface created when a process is first started, and used to give and get access to other interfaces (\doxyref{Node}{p.}{classinterfaces_1_1_node}, \doxyref{Chain}{p.}{classinterfaces_1_1_chain}, \doxyref{Wallet}{p.}{classinterfaces_1_1_wallet}, etc).

There is a different \doxyref{Init}{p.}{classinterfaces_1_1_init} interface implementation for each process (bitcoin-\/gui, bitcoin-\/node, bitcoin-\/wallet, bitcoind, bitcoin-\/qt) and each implementation can implement the make methods for interfaces it supports. The default make methods all return null.
