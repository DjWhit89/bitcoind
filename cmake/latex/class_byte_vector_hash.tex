\doxysubsection{Detailed Description}
Implementation of Hash named requirement for types that internally store a byte array. This may be used as the hash function in std\+::unordered\+\_\+set or std\+::unordered\+\_\+map over such types. Internally, this uses a random instance of Sip\+Hash-\/2-\/4.
