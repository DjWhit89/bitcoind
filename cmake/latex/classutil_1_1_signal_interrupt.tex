\doxysubsection{Detailed Description}
Helper class that manages an interrupt flag, and allows a thread or signal to interrupt another thread.

This class is safe to be used in a signal handler. If sending an interrupt from a signal handler is not necessary, the more lightweight \doxyref{CThread\+Interrupt}{p.}{class_c_thread_interrupt} class can be used instead.
