\doxysubsection{Detailed Description}
Double ended buffer combining vector and stream-\/like interfaces.

\begin{quote}
\begin{quote}
and $<$$<$ read and write unformatted data using the above serialization templates. Fills with data in linear time; some stringstream implementations take N$^\wedge$2 time. \end{quote}
\end{quote}
