is a home for public enum and struct type definitions that are used by internally by wallet code, but also used externally by node or GUI code.

This file is intended to define only simple types that do not have external dependencies. More complicated public wallet types like \doxyref{CCoin\+Control}{p.}{class_c_coin_control} should be defined in dedicated header files.
