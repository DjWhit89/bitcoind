\doxysubsection{Detailed Description}
Formatter for undo information for a \doxyref{CTx\+In}{p.}{class_c_tx_in}

Contains the prevout\textquotesingle{}s \doxyref{CTx\+Out}{p.}{class_c_tx_out} being spent, and its metadata as well (coinbase or not, height). The serialization contains a dummy value of zero. This is compatible with older versions which expect to see the transaction version there.
