\doxysubsection{Detailed Description}
xoroshiro128++ PRNG. Extremely fast, not appropriate for cryptographic purposes.

Memory footprint is very small, period is 2$^\wedge$128 -\/ 1. This class is not thread-\/safe.

Reference implementation available at {\texttt{https\+://prng.\+di.\+unimi.\+it/xoroshiro128plusplus.\+c}} See {\texttt{https\+://prng.\+di.\+unimi.\+it/}}
