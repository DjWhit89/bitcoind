\doxysubsection{Detailed Description}
class\+: \doxyref{OSError}{p.}{classsubprocess_1_1_o_s_error} Thrown when some system call fails to execute or give result. The exception message contains the name of the failed system call with the stringisized errno code. Check \doxyref{Popen}{p.}{classsubprocess_1_1_popen} class API\textquotesingle{}s to know when this exception would be thrown. Its usual that the API exception specification would have this exception together with \doxyref{Called\+Process\+Error}{p.}{classsubprocess_1_1_called_process_error}.
